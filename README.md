# Quote Classifier

- Classifies Arabic quotes by topic  
- Python machine learning model  (using Tensorflow)
- Flask backend serving the model  (FastAPI)
- React.js frontend interface  
- User inputs an Arabic quote and receives a predicted category  
- Demo recording included
